Unpacking APPSPACK

  • To unpack APPSPACK, type

    tar -xzvf appspack-4.0.2.tar.gz

    or

    gunzip -c appspack-4.0.2.tar.gz | tar -xvf -

    This will create a directory named appspack-<version> containing the source code for APPSPACK.

  • In general, you may wish to rename this directory to appspack; i.e.,

    mv appspack-4.0.2 appspack

    Throughout the remainder of the documentation, we will assume that APPSPACK is installed in the directory appspack.

  • To unpack the documentation, type

    mv appspack-4.0.2-htmldoc.tar.gz appspack/doc
cd appspack/doc
gunzip -c appspack-4.0.2-htmldoc.tar.gz | tar -xzf -

    Note that the images will not appear correctly unless the documentation is put in the right location.

  • To view the documentation, point your browser to appspack/doc/html/index.html
