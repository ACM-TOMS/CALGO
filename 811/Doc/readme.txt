The NDA collection contains the following files:

1) File  psubs.f  containing optimization subroutines.
2) File  mbsubs.f containing matrix subroutines.
3) File  tbsubs.f containing test subroutines.
4) Files tminu.f, tminl.f, tbunu.f, tbunl.f, 
         tnewu.f, tnewl.f, tvaru.f, tvarl.f, containing 
         sample routines for the demonstration of the NDA
         subroutines.
5) Files pmin.txt, pbun.txt, pnew.txt pvar.txt containing
         descriptions of the basic NDA subroutines.
6) File  com which is a batch file for starting the sample 
         routines.
