Initializes the intra-\/node execution environment. 

The interface function initializes Star\+PU (and cu\+B\+L\+AS) and pauses all worker The {\ttfamily cores} argument specifies the {\bfseries{total number of used C\+PU cores}}. In distributed memory mode, one C\+PU core is automatically allocated for the Star\+P\+U-\/\+M\+PI communication thread. One or more C\+PU cores are automatically allocated for G\+PU devices.
