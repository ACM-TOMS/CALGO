Structure of the Toolbox IPscatt

This file shows the structure of the toolbox IPscatt, i.e. dependencies of functions. The structure of files in folders is shown in ../index.html.

Please note that this structure represents the most important dependencies of functions in case of typical work flows and therefore is not complete.

We distinguish between internal and external functions. The external ones are marked in this overview by [external] to make clear that they are more or less easy to use outside the typical work flow of IPscatt.
