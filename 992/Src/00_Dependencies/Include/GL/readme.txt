-------------------------
Only on Windows platforms
-------------------------

Download and extract the Windows-binaries

glew-x.y.z-win32.zip

of the function library •GLEW• from the link

http://glew.sourceforge.net/

where x.y.z denotes its current version number,
then perform the following copying operation:

glew-x.y.z/include/GL/glew.h --> 00_Dependencies/Include/GL/glew.h
