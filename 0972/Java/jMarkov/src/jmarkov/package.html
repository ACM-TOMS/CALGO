Provides the basic elements to model continuous time Markov chains (CTMC). An application should extend the class {@link jmarkov.MarkovProcess}, {@link jmarkov.basic.State} and, optionaly the class {@link jmarkov.basic.Event}. The package also includes facility to model Matrix Geometrix Systems. For overviews, tutorials, examples, guides, and tool documentation, please see:

  • JMarkov User's Manual.
  • COPA group WEB page.
