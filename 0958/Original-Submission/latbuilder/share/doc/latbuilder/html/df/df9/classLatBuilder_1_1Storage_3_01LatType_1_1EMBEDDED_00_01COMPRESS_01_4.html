Detailed Description

template<Compress COMPRESS>
class LatBuilder::Storage< LatType::EMBEDDED, COMPRESS >

Vector permutation for embedded lattices.

The vector embedded at level \(m\) in base \(b\) consists of the first \(b^m\) components of the internal representation of the vector (the lower the level, the deeper the embedding).

The indices are ordered by level. Thus, refining a logical vector by adding a new level op components can be achieved by appending elements at the end of the physical vector. Within each level, a permutation given by the inverse cyclic subgroup corresponding to that level is applied.

Template Parameters
COMPRESSCompression type (either None or Symmetric). If Compress::SYMMETRIC, the permuted indices are compressed by assuming that the vector components at unpermuted indices \(i\) and \(b^M - i\) have the same value, for all \(i = 0, \dots, b^M - 1\), where \(M\) is the maximum level of embedding.

See the output of Storage.cc for an illustration of this.
