iisignature version 0.24
========================

The main source code is in iisignature-0.24-src-and-examples/.
Its documentation is in doc/.
Code for the experiments is in results/.
Other code referred to in the paper is in other_code. 

It is assumed that you have Python and numpy installed and that the Python distutils tools can access a compiler.  

To run the tests (which are the basic examples) go to iisignature-0.24-src-and-examples and type
   python setup.py test
