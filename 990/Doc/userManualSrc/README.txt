The authors provided two user guides:

1. TOMSUserGuide: this is a manual for the software package
which was submitted and accepted for publication in ACM TOMS. It
contains the library of computational routines.

2. CompleteUserGuide: this is a manual for the complete system
including te GUI that was prepared to present the results
of the computations to the user in a graphical format. This
is NOT included in this software package but is included for
completeness.
