How To Cite APPSPACK

Thank You!

Thank you for using APPSPACK. If you are using version 4.0 or later, we ask that you please cite the following in any resulting publications or reports.

  • Genetha A. Gray and Tamara G. Kolda, APPSPACK 4.0: Asynchronous Parallel Pattern Search for Derivative-Free Optimization, Technical Report SAND2004-XXXX, Sandia National Laboratories, Livermore, CA, August 2004. (Submitted to ACM TOMS for publication.)

  • Tamara G. Kolda, Revisiting Asynchronous Parallel Pattern Search for Nonlinear Optimization, Technical Report SAND2004-8055, Sandia National Laboratories, Livermore, California, February 2004. (In revision for SIAM J. Optimization.)

If you are using an earlier version, please cite the following.

  • Patricia Hough, Tamara G. Kolda, and Virginia Torczon, Asynchronous Parallel Pattern Search for Nonlinear Optimization, SIAM J. Scientific Computing, 23(1):134-156, June 2001.

  • Tamara. G. Kolda and Virginia J. Torczon, On the Convergence of Asynchronous Parallel Pattern Search, SIAM J. Optimization, 14(4):939-964, May 2004

Other Resources

For links to the papers listed above and other related work, see Publications about APPSPACK.
