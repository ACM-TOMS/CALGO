Detailed Description

APPS Communicator - An interface for both PVM and MPI commands.

Note:
Although GCI still supports PVM, it is no longer officially supported by APPSPACK. If you would like to use PVM, please contact Tamara G. Kolda.
Author:
MPI interface contributed by H. Alton Patrick, Summer 2000.
